TwigInflection
==============

pluralize
---------


``` jinja

{{ "chicken"|pluralize }}

{{ rows.length }} {{ "row"|pluralize(rows.length) }}

{{ users.length }} {{ "person"|pluralize(users.length, "users") }}

```
